const
   用于放置在参数、全局变量和局部变量的前面
   const T v;
   T const v; 同前一种。
   表示 v 的数值只读。
   用于修饰指针的时候，就有两种不同的情况。
   1) const T *v;
   T const *v; 同前一种。
   2) T *const v;
   3) const T *const v;
   解释：
   1) 表示 v 指向的变量的数值只读。
   2) 表示 v 的指向不可变。
   3) 结合 1) 和 2)。
   如果修饰在数组上，
   它表示数组的每一个元素不可变。

static
1. 修饰在临时变量上，
   表示该变量只分配一次内存。
2. 修饰在全局变量和函数上，
   表示当前信息只在当前起效，
   出了文件就访问不了了。
3. 修饰在数组参数的 [] 里，
   表示数组至少这么多个元素。

inline（C99）
只能修饰在函数上，表示函数尽量在编译期间
被编译器内联到调用方。

register
用于变量上，表示当前变量尽量放在
寄存器里，以便节约计算时间和增强
运算性能。

restrict（C99）
用于指针参数上（至少两个参数，且参数类型一致）。
表示这些指针参数存储的地址数值肯定是不同的。

extern
1. 用于标记函数，可以导出为 dll 文件，
   且文件导出后（dll 形式）仍可通过工具
   查看到当前函数。
2. 用于标记全局变量或临时变量，表示该
   变量来自于别的文件。
